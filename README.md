# Aadhaar Sentinel

An intelligent command center for identifying anomalies and patterns in Aadhaar operations.

## Features
*   **Data Aggregation**: Merges Biometric, Demographic, and Enrolment data.
*   **Anomaly Detection**: Uses Isolation Forest.
*   **Interactive Dashboard**: Streamlit-based UI.

## Usage
Run the dashboard:
```bash
python -m streamlit run src/app.py
```
